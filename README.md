# تعلّ — The Weight of Letters

A computational investigation into whether each Arabic letter in the Quran carries intrinsic semantic significance — a discoverable "meaning direction" grounded in four independent layers of evidence, verified by the text's own usage.

## لسان not لغة

The Quran describes itself as لسان عربي — an Arabic *tongue*, not لغة عربية — an Arabic *language*. This project honors that distinction. The text is analyzed through the body's articulatory geography and the text's own internal patterns — never through the linguistic system built around it.

## The Four Layers

1. **Sonic (الصوت)** — Where in the body each letter originates (7 articulatory zones) and how diacritics modulate it
2. **Geometric (الشكل)** — Each letter as a transformation of alif; visual families; dot differentiation
3. **Structural (الرسم)** — Identity, position, morphological role within the text
4. **Frequency (التكرار)** — Luminosity: count, distribution, regularity, rank, evolution across the text

## The Loop

Opens with الم — three letters placed before the claim of no discrepancy. Closes by returning to الم after all 28 semantic directions are independently established from ordinary vocabulary.

The text speaks. We measure. The letters will tell us whether they carry weight.
